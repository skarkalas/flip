Bitwise Operators

These operators operate on numeric operands. In a bitwise operation the operands are treated as signed 32-bit integers in big-endian order and in twoâ€™s complement format. The operation takes sequences of 32 bits and returns a normal numerical value.

The available operators follow: 


And			&
Or			|
Xor			^
Not			~
Left shift		<<
Right shift		>>
Zero-fill right shift	>>>
And

This is a binary operator. It compares the operands bit by bit and returns 1 for each pair where both values are equal to 1.

Or

This is a binary operator. It compares the operands bit by bit and returns 1 for each pair where either or both values are equal to 1.

Xor

This is a binary operator. It compares the operands bit by bit and returns 1 for each pair where either but not both values are equal to 1.

Not

This is a unary operator. It inverts all the bits of the operand.

Shift left

This is a binary operator. The right operand is treated as a binary value. It shifts the bits of its left operand to the left for as many places as given in the right operand. The bits shifted off at the leftmost side are discarded and zeroes are shifted in from the right.

Shift right

This is the opposite of the previous operation. It shifts the bits of its left operand to the right for as many places as given in the right operand. The bits shifted off at the rightmost side are discarded. This operation preserves the sign because it keeps the most significant bit unchanged.

Zero-fill shift right

This is the same operation as the previous one. The difference is that it does not preserve the sign as it shifts in zeroes from the left.
