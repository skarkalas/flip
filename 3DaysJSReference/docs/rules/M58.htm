Unnecessary use of 'else'

The following code is valid and works as expected: 


function f(a)
{
    if(a < 5)
    {
        return 0;
    }
    else
    {
        return 1;
    }
}

The only problem is that the use of else is pointless since there is a return statement in the preceding block. If the condition is evaluated to true, the function returns and there is no chance to execute the alternative case anyway. The following is better: 


function f(a)
{
    if(a < 5)
    {
        return 0;
    }

    return 1;
}
