A variable is expected to be declared but it is not

The assignment cannot be used instead of the comma operator in a multiple declaration statement. The following code is valid but conveys a potential misunderstanding: 


var x = y = z = 10;

Only x is declared. The remaining two variables are only assigned values. If this is intended to be a multiple assignment statement then the following is needed: 


var x, y, z;
x = y = z = 10;
