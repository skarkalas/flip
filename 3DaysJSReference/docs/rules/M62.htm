A variable is declared in a block

Variables in JavaScript have function scope. Declaring a variable in a block does not restrict its scope within that block. The following is permissible by the language but doesn't mean that the variable i is not going to be available outside of the loop. 


for (var i = 0; i < 5; i++)
{
	alert(i);
}

It is therefore better to prevent potential misunderstandings and declare the iterator outside of the loop. 


var i;
for (i = 0; i < 5; i++)
{
	alert(i);
}
