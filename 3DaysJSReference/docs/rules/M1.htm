Variable Redeclaration

There is a variable that has been declared multiple times within a function. The same variable identifier has been used in many declarations. 


function f()
{
	var x = 4;
	var x = 6;	
}

function f()
{
	var x = 4, x = 6;	
}

The problem in this case is that you intended to declare two variables instead of one. The system, though, creates only one and assigns two values to it. The value of the final assignment is retained. You can fix the problem by renaming one of the variables. 


function f()
{
	var x = 4, y = 6;	
}

If the intention is to simply assign a new value to the same variable then use a semicolon to create two statements (second case). 

	
function f()
{
	var x = 4;
	y = 6;	
}
