A name is used both as an identifier and a label

There is a name in the code that is used both as a variable identifier and a label. 


var x = 7;		//identifier

function f()
{
    x:			//label

    while(x > 0)	//identifier
    {
        x--;		//identifier
        if(x === 5)	//identifier
        {
            break x;	//label
        }
    }
}

In terms of syntax there is no problem as there is no name clash. Labels exist in a completely separate namespace. The problem is readability and clarity of code. The solution is to rename one of the two. 


var x = 7;		//identifier

function f()
{
    y:			//label

    while(x > 0))	//identifier
    {
        x--;		//identifier
        if(x === 5)	//identifier
        {
            break y;	//label
        }
    }
}
