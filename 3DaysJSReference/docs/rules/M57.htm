A variable has not been initialised on declaration

If a variable is declared but not initialised it takes the value (and type) of undefined. 


function f()
{
    var a;
    a = 1;
    return a;
}

That might not be what the programmer intended to do and it is a potentially problematic statement. It is better practice to always initialise the variables on declaration. If there is no available value then the value null can be given. 


function f()
{
    var a = 0;
    a = 1;
    return a;
}
