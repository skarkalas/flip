Unexpected break statement

There is a break statement in a loop that will terminate it after the first iteration. 


var x = 7;

while(x > 0)
{
	x--;
	break;  
}

Statements like break (or continue) should always be wrapped in a decision structure. 


var x = 7;

while(x > 0)
{
	x--;
	if(x === 5)
	{
		break;
	}
}
