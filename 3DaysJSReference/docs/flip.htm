What is FLIP Learning?

FLIP Learning (Flexible Intelligent Personalised Learning) is an exploratory learning environment that teaches JavaScript. It offers the following functionality: advanced I/O and debugger, 2D graphics, code quality control, intelligent tutoring support, automatic assessment, alternative code representations, code tracing visualisations and automatic refactoring. Automatic assessment and alternative code representations have not been implemented yet. Automatic refactoring may be available depending on the type of problem.
