Logical Operators

These operators ideally operate on Boolean values and they also return a Boolean value. It is possible to operate on other types and use their Boolean equivalent after the conversion.

The available operators follow: 


not	!
and	&&
or	||
Not

The not operator is unary whereas the other two are binary. Not negates the truth-value of its operand.

And

The only case in which this operator returns true is when both of its operands are evaluated to true. If either (or both) of them is false then it returns false.
